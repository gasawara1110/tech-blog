# はじめに

先日、歩行ロボットでよく使われる **QDD Actuator** についてLTをしました。

QDDは **Quasi Direct Drive** の略で、日本語では「準ダイレクトドライブ」と呼ばれることが多いアクチュエータ構成です。高トルク密度のBLDCモータに、低減速比の減速機を組み合わせた構成で、近年の四足歩行ロボットやヒューマノイドロボットでよく見かけるようになってきました。

この記事では、LTスライドをベースにしつつ、スライドだけでは説明しきれなかった部分を短く補足していきます。

![](/images/20260720-qdd-actuator-for-legged-robots/slide-01.png)

この記事で扱う内容は、主に以下の5点です。

1. なぜ今 歩行ロボ / QDD なのか
2. QDDアクチュエータとは何か
3. 従来型高減速比アクチュエータとの違い
4. 低減速比の利点
5. 歩行ロボットでQDDが重要な理由

なお、本記事は公開情報と筆者の理解に基づく整理です。市場規模やコスト構成などの数値は、参照元によって前提が異なるため、厳密な比較値というより「なぜアクチュエータが重要部品として見られているのか」を理解するための材料として扱っています。

# 1. なぜ今 歩行ロボ / QDD なのか

![](/images/20260720-qdd-actuator-for-legged-robots/slide-02.png)

ここ数年、ヒューマノイドや四足歩行ロボットへの注目がかなり高まっています。

その背景には、AI技術の進展だけでなく、モータ、減速機、バッテリ、エッジコンピュータ、センサ、シミュレーション環境など、ロボットを構成する要素技術が同時に進化していることがあります。

その中でもアクチュエータは、ロボットの性能・コスト・制御性を大きく左右する中核部品です。

ロボットが実際に外界へ力を出す部分は、最終的にはアクチュエータです。AIがどれだけ賢くなっても、関節が重い、硬い、熱い、壊れやすい、力制御しにくい、といった問題があると、ロボット全体の性能はそこで頭打ちになります。

そのため、歩行ロボットを考えるうえでは「どんなアクチュエータを使うか」がかなり重要になります。

# 2. QDDアクチュエータとは何か

![](/images/20260720-qdd-actuator-for-legged-robots/slide-03.png)

QDDアクチュエータは、ざっくり言うと **高トルク密度モータ + 低減速比ギア** で構成されたアクチュエータです。

一般的なロボット用アクチュエータでは、モータ単体のトルクを減速機で大きく増幅して使います。高減速比にすると、小型モータでも大きな出力トルクを得やすくなります。

一方、QDDでは減速比を低めに抑えます。そのぶんモータ側には高いトルク密度や大きな電流を扱う能力が求められますが、出力側から見た摩擦や反射慣性を小さくしやすくなります。

最近のQDDアクチュエータでは、モータ側だけでなく出力側にもエンコーダを持つデュアルエンコーダ構成の製品も増えています。これにより、減速機のたわみやバックラッシを含めた出力軸側の状態をより直接的に扱いやすくなります。

歩行ロボット、外骨格、協働ロボットのように、人や環境との接触が重要になるロボットでは、この「出力側の力や動きの扱いやすさ」が大きな意味を持ちます。

# 3. 従来型高減速比アクチュエータとの違い

![](/images/20260720-qdd-actuator-for-legged-robots/slide-04.png)

従来型の高減速比アクチュエータは、小型モータから大きなトルクを取り出しやすく、静的な保持や高精度な位置決めに強いという利点があります。

その一方で、減速比が大きくなるほど、減速機内部の摩擦や反射慣性の影響が大きくなりやすくなります。  
その結果、出力軸トルクをモータ電流だけから正確に推定するのが難しくなります。

そのため、高精度な力制御が必要な場合には、出力軸側にトルクセンサや力覚センサを追加することがあります。  
ただしトルクセンサは高価になりやすく、機械構成や配線も複雑になります。

また、ハーモニックギアのような高減速比減速機はコンパクトで高トルク化しやすい一方、衝撃荷重には注意が必要です。  
私自身も過去に、ロボットアームの評価中にハーモニックギアを壊してしまい、出力軸が回らなくなった経験があります。

歩行ロボットのように接地衝撃や外乱を受ける用途では、このような経験からも、バックドライブ性や衝撃を逃がしやすい構成の重要性を感じています。

QDDはその逆で、減速比を低くすることで、バックドライブしやすく、外力を逃がしやすく、力制御と相性が良い構成を狙います。

もちろん、QDDにも弱点があります。

低減速比にするぶん、同じ出力トルクを出すにはモータ側に大きな電流が必要になりやすく、発熱や電源設計が厳しくなります。また、静的に大トルクを保持し続ける用途では、高減速比アクチュエータの方が向いている場合もあります。

つまり、QDDは万能というより、**接触・力制御・動的動作を重視するロボットに向いたアクチュエータ構成** と考えるのがよさそうです。

# 4. 低減速比の利点

![](/images/20260720-qdd-actuator-for-legged-robots/slide-05.png)

低減速比の大きな利点は、摩擦と反射慣性の影響を抑えやすいことです。

減速機には、静止摩擦、クーロン摩擦、粘性摩擦、Stribeck効果など、さまざまな摩擦要素があります。特に低速域では摩擦が非線形になりやすく、「動き出しが重い」「一度動くと軽くなる」といった挙動が出ることがあります。

この摩擦は、トルク推定にも影響します。

理想的には、BLDCモータの出力トルクは電流から推定できます。

```text
出力軸トルク ≒ 効率 × 減速比 × トルク定数 × 電流
```

しかし実際には、減速機内部の摩擦や損失があるため、電流から推定したトルクと実際の出力軸トルクにはずれが生じます。

また、反射慣性は減速比の2乗に比例して出力軸側へ効いてきます。

```text
出力軸から見た反射慣性 ≒ 減速比^2 × モータ慣性
```

そのため、減速比が大きいアクチュエータでは、出力軸側から見たモータ慣性が大きくなり、バックドライブしにくくなります。

QDDは減速比を低く抑えることで、摩擦・非線形性・反射慣性の影響を抑えやすく、低速域の力制御や接触適応で扱いやすくなります。

# 5. 歩行ロボットでQDDが重要な理由

![](/images/20260720-qdd-actuator-for-legged-robots/slide-06.png)

歩行ロボットの関節には、かなり複雑な要求が同時にかかります。

たとえば歩行中には、接地の瞬間に衝撃が入り、支持期には体重を支えるための持続的なトルクが必要になります。蹴り出しでは大きなトルクとパワーが必要になり、遊脚期では脚を素早く振り出す速度応答も必要です。

つまり歩行ロボットの関節には、以下のような性質が同時に求められます。

- 接地衝撃に対する適応性
- 外力を受けたときの安全性
- 力制御のしやすさ
- 低速高トルク動作
- 短時間のピークパワー
- 遊脚期の速度応答

このような用途では、単に「位置を正確に決める」だけでなく、環境と接触しながら力をうまく扱うことが重要になります。

QDDは、低減速比によるバックドライバビリティの高さ、反射慣性の小ささ、力制御との相性の良さから、歩行ロボットの関節アクチュエータとして有力な選択肢になっています。

特に接地の瞬間には、関節に意図しない外力や衝撃が入ります。  
このとき、アクチュエータが硬すぎると機構や減速機に負担が集中しやすくなります。  
QDDのように低減速比でバックドライブしやすい構成は、こうした接触を含む運動で扱いやすいと考えています。

# おわりに

QDDアクチュエータは、単に「低減速比のモータユニット」というだけではなく、歩行ロボットに必要な接触適応、力制御、速度応答を支える重要な要素だと考えています。

一方で、大電流、発熱、電源設計、静的保持トルクなどの課題もあり、すべての用途で高減速比アクチュエータより優れているわけではありません。

今後は、MuJoCoなどのシミュレーションを使って、歩行時の関節トルク・速度・パワーを見ながら、QDDや回生エネルギーについてもう少し掘り下げていきたいと思います。
